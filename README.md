# 📖 整合包介绍

踏入蔚蓝边界，体验 Enigmatica 对经典空岛玩法的全新演绎：空岛。

Enigmatic Skies是一款以流程推进为核心的整合包，目标看似简单，但实现过程绝不会轻松——你将在这片寒冷虚空中艰难求生，而水资源更是极其有限。

在这个没有怪物的世界里，真正的威胁只有严寒本身；如何维持体温将成为你整个冒险过程中持续面对的挑战。
利用你的炼金知识，赋予村民生命，并通过自定义交易让他们协助你的发展。
将各种生物——从普通动物到恐怖怪物——进行嬗变转化，以获得建造复杂机械与解锁更强大魔法所需的材料。

最终，你将掌控创造本身的力量，让所有努力汇聚于终极目标：创造“世界之种”。

- 使用冷汗提供的环境冷热机制。保持温暖，否则只能迎接死亡！
- 为村民提供复杂餐食组合，解锁他们完全自定义、且别处无法获得的特殊交易。
- 享受自动化村民交易：持续喂养村民，让他们源源不断地提供物资。
- 借助新生魔艺的强力法术，制造物品并与友好的探宝蟹进行交易。
- 结合神秘学、灵灾、基因与末影接口进行自定义怪物嬗变。
- 使用现代工业化构建复杂工业能源体系，从基础蒸汽锅炉一路发展到幻想级聚变能源。
- 通过云层、西瓜，甚至循环废料来制造水资源，以维持能源基础设施与资源生产链运作。
